# Java Base Networking

## Overview
A foundational repository establishing core networking competencies in Java. Validates understanding of low-level networking abstractions before migrating to higher-level frameworks like Spring.

## Technical Scope
- **Raw Socket Manipulation:** Direct implementation of `ServerSocket` and client `Socket` interaction, establishing synchronous bidirectional TCP/IP communication channels via `DataInputStream` / `DataOutputStream`.
- **HTTP Protocol Implementation:** Manual HTTP server and client architecture demonstrating packet structuring and payload execution.
- **Servlet Architecture:** Implementation of standardized Java Servlets handling stateless request-response web models.
- **Build Initialization:** Managed through a Maven build lifecycle (`pom.xml`).
